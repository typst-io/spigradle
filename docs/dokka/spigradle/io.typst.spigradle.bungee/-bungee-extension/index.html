BungeeExtension

open class BungeeExtension(project: Project)(source)

Bungeecord configuration for the 'plugin.yml' description, and debug settings.

Groovy Example:

spigot {
  author 'Me'
  depends 'ProtocolLib', 'Vault'
  debug {
    agentPort 5005
  }
}
